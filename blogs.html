Question-1: What are the different ways to select an element in the DOM?

  1. Selecting by ID: document.getElementById(id)
  2. Selecting by Class Name: document.getElementsByClassName(className)
  3. Selecting by Tag Name: document.getElementsByTagName(tagName)
  4. Selecting with CSS Selectors (First): document.querySelector(selector)
  5. Selecting with CSS Selectors (All): document.querySelectorAll(selector)

Question-2: What is the difference between innerHTML, innerText, and textContent?

  1. innerHTML: Gets or sets HTML including descendants in an element.
  2. innerText: Gets or sets the visible text content of an element, it returns text that is actually rendered on the page.
  3. textContent: Retrieves the complete text content, regardless of rendering.

Question-3: What is event delegation in the DOM?

Event delegation is a technique to handle events from a parent for all of its children, rather handling for each child individually. It helps to handle events, even when some children are added later into the parent.

Question-4: What is event bubbling in the DOM?

Event bubbling describes how an event travel through the DOM tree. It starts from the target element and travels to its parent, up to the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

  1. Create Elements: document.createElement(tagName)
  2. Add Elements: parentElement.appendChild(childElement), parentElement.insertAdjacentHTML(position, newElement)
  3. Remove Elements: parentElement.removeChild(childElement), parentElement.replaceChildren() (Removes All)
